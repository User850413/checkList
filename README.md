## 개요

체크리스트!

### 구현된 기능

- 새 리스트 추가, 수정, 삭제
  - 리스트 내 항목 추가, 수정 삭제
- 로그인 및 회원가입

### 기능 필요사항

- 리스트 입력 후 체크 가능하게
  - 체크 후 사라지지 않음
- 입력된 리스트 삭제 가능해야함
- 반복 삽입 가능했으면 좋겠음
- 회원 탈퇴 가능하게
- 리스트 색상 변경 가능

### 기능 추가사항

- 항목이 언제 업데이트 되었는지
- 마이페이지가 있었으면 좋겠음
  - (하루 지난) 완료 체크리스트 확인 가능
  - 간단한 메모가 됐으면 좋겠다
- 일단 완료하지 못한 체크리스트는 삭제되게
- 클릭 후 드래그로 리스트 순서가 바뀌었으면
- 퍼센트 리스트도 있었으면

_생각나는 게 있으면 계속 추가하기_
