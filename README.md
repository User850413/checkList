## 개요(11/19)

체크리스트!

### 기능 필요사항

- 리스트 입력 후 체크 가능하게
  - 체크 후 사라지지 않음
- 입력된 리스트 삭제 가능해야함
- 반복 삽입 가능했으면 좋겠음

### 기능 추가사항

- 마이페이지가 있었으면 좋겠음
  - (하루 지난) 완료 체크리스트 확인 가능
  - 간단한 메모가 됐으면 좋겠다
- 일단 완료하지 못한 체크리스트는 삭제되게

_생각나는 게 있으면 계속 추가하기_

### 기술 스택

- Next App Router
- Tailwind
